# Neuroflow

A visual IDE for building and training machine learning models with a node-based interface.

## Features

- Visual node-based workflow editor
- CSV data loading and preprocessing
- Multiple regression models (Linear, Multi-Linear, Logistic)
- Data visualization and model evaluation
- Real-time model training and predictions

## Development

### Frontend

```bash
npm install
npm run dev
```

### Backend

See [backend/README.md](backend/README.md) for backend setup instructions.

### Electron App

```bash
npm run electron:dev
```

## Building

```bash
npm run build
npm run electron:build
```
